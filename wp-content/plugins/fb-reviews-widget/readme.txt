== Installation ==

1. Upload plugin files to `/wp-content/plugins/` or install via the Plugins screen.
2. Activate **Reviews Widget for Google, Yelp & Recommendations**.
3. Go to **Social Reviews** in wp-admin.
4. Connect your Google, Yelp and Facebook pages.
5. Add the widget, shortcode or block anywhere.

== FAQ ==

### ‚ùì How many reviews can I display?

Google: up to 10, via public API.
Yelp: up to 3, via public API.
Facebook: all Recommendations for your own Pages.

### ‚ùì Does the plugin use official APIs?

Yes. Google and Yelp use public endpoints.
Facebook requires login authorization and uses the official Graph API.

### ‚ùì How many business locations can I connect?

Unlimited ‚Äî you can connect any number of locations.

### ‚ùì How many widgets can I create?

Unlimited ‚Äî each widget can have its own layout and source combination.

### ‚ùì Can I hide specific reviews manually?

Yes. You can hide or show any review.

### ‚ùì What layouts are supported?

List, grid and slider (carousel).

### ‚ùì Can I combine different platforms into one feed?

Yes ‚Äî Google + Yelp + Facebook in any combination.

### ‚ùì Do I need a Facebook app?

No. Simply authorize with your Facebook account.

### ‚ùì Is JavaScript required?

Only for ‚ÄúRead more‚Äù, slider and pagination.
